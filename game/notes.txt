Dialogue:
Line:
- Show text/speaker
- If 1 child, wait for any input & get next node
- If >1 child:
    - Show edges as responses
    - Select edge based on input

Branch:
- On enter, eval first edge
- Set node to edge target if condition true
- If false, continue evaluating edges until true or last edge

Group:
- Replace with (evaluated) outgoing edges when it appears as valid edge child
